# MongoChatApp

Simple chatting application utilizing socket.io. Messages are stored in a MongoDB database.

### TECHNOLOGIES
1. Node.js
2. MongoDB
3. Socket.io

### WHAT I LEARNED
1. How to emit events from the client and how to receive them on the server
